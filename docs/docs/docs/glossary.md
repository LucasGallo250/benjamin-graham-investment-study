# 📚 Glossary of Key Terms

## Intrinsic Value
The true value of a company independent of market price.

## Margin of Safety
The difference between intrinsic value and market price.

## Mr. Market
A metaphor representing market emotional volatility.

## Defensive Investor
An investor focused on safety and stability.

## Enterprising Investor
An investor willing to dedicate time to deeper analysis.

## Current Liquidity Ratio
Current Assets / Current Liabilities (recommended ≥ 2.0)

## P/E Ratio
Price divided by earnings (recommended ≤ 15)

## P/B Ratio
Price divided by book value (recommended ≤ 1.5)
